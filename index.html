Using Notes

Tap New note for a standalone study note, or use a verse’s note button to attach a note directly to it. Add comma-separated tags to organise a topic.

Cross-references can connect Quran verses, Bible passages, and hadith. Tap a reference on a note to read it, or use View references in the editor to compare them together.

Search: words find any match, "exact phrase" keeps words together, and mercy + prayer requires both.

Shared notes require an account

Sign in to securely view or edit notes shared across accounts.

Searching the library

Choose books with Search in, then search any downloaded Quran translation, tafsir, Bible book, commentary, or hadith collection. Results appear as each book finishes, so you can open or select them immediately.

"eternal life" finds an exact phrase. mercy + forgiveness requires both terms in the same passage. Use Select results to collect several passages into one note.
